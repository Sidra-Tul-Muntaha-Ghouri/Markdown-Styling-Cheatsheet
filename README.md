# Markdown-Styling-Cheatsheet

## Headings:
```
# Heading 1
## Heading 2
### Heading 3
```
### OutPut:

# Heading 1
## Heading 2
### Heading 3

## Links
- Link with Text:
```
[link Text](link address)
```
### Output:
[LinkedIn](https://www.linkedin.com/in/sidra-tul-muntaha-ghouri/)
- Direct Email Link:
  
```
[Email](mailto:sidratulmuntaha135@gmail.com)
```
### Output:
[Email](mailto:sidratulmuntaha135@gmail.com)

## More than 1 Line Space

```
Line 1
\
\
Line 2
```
### Output:

Line 1
\
\
Line 2

## Code
```
`inline code`
```
### Output:
`inline code`

## Task List
```
- [x] Task 1
- [ ] Task 2
- [ ] Task 3
```
### Output:
- [x] Task 1
- [ ] Task 2
- [ ] Task 3
